# Tower Defense Game

## Overview
This project is a Tower Defense Game developed in C++ using the SFML graphics library.  
The game includes enemies, towers, waves, player interactions, and map rendering using object-oriented programming concepts such as inheritance, polymorphism, abstraction, and dynamic memory management.

---

# How to Compile and Run the Project

## Requirements
Before compiling the project, make sure the following are installed:

- C++ Compiler (G++ or Visual Studio)
- SFML Library
  - Visual Studio
